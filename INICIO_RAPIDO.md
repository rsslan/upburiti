# 🚀 INÍCIO RÁPIDO

## 3 PASSOS PARA RODAR

### 1. Instalação (Uma única vez)
```bash
npm install
npm run build
```

### 2. Iniciar COM CMD OCULTO ⭐
```
Duplo clique em: iniciar_oculto.vbs
```

### 3. Acessar no Celular
```
1. Execute: Obter_IP_Celular.bat
2. Note o IP (ex: 192.168.1.100)
3. No celular: http://192.168.1.100:5173
```

---

## COMPORTAMENTO

✅ **CMD fica oculto** - não aparece nenhuma janela  
✅ **Server fecha automaticamente** quando:
   - Você fecha a aba do navegador
   - Fica 10 minutos inativo

✅ **Mobile** - Funciona perfeitamente em:
   - Android (Chrome, Firefox, etc)
   - iPhone (Safari, Chrome, etc)

---

## SE ALGO DER ERRADO

Abra cmd nesta pasta e tente:
```
npm install
npm run build
npm run dev
```

Depois acesse: http://localhost:5173
